# Img Slicer

A wee script slicing images along detected contours.

## Execution

### crop.py
```
Optional arguments:
  -h, --help  show this help message and exit

Required arguments:
  -s FILE     input image file
  -d FILE     output folder (will be created if non-existent)
```

### gui.py
Fires up a simple tkinter GUI
